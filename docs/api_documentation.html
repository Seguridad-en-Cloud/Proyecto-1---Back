Render Menu

qr

Get Qr Code

Generate QR code for the authenticated user's restaurant menu.

The QR encodes the public menu URL (/m/{slug}). The code is static -- it always points to the same URL, but the menu content updates in real time.

Supports color customization (RF-20) and named sizes S/M/L/XL (CU-07).

Args: user_id: Authenticated user ID. session: Database session. format: Output format ('png' or 'svg'). size: Named size preset (S/M/L/XL). fill_color: QR foreground color. back_color: QR background color.

Returns: QR code image (PNG or SVG).

Authorizations:
HTTPBearer
query Parameters
format
string (Format) ^(png|svg)$
Default: "png"
size
string (Size) ^(S|M|L|XL)$
Default: "M"

QR output size: S (200px), M (400px), L (800px), XL (1200px)

fill_color
string (Fill Color) <= 20 characters
Default: "black"

QR foreground color (hex like '#FF0000' or CSS name like 'black')

back_color
string (Back Color) <= 20 characters
Default: "white"

QR background color (hex like '#FFFFFF' or CSS name like 'white')

Responses

/api/v1/admin/qr

Response samples

  • 200
  • 422
Content type
application/json
null
